== Interview

这个简单的系统目前分为以下几部分：

1） 用户基本注册、登陆、在线统计功能

2） 游客的保存、在线统计功能

3） 用户、游客的全部信息api

实现方式主要是通过cookie的方式，保存注册用户、游客的信息，便于记录他们的具体操作信息(登陆、登出)

ps: 注册用户的在线时间以点击‘退出’为准
    
    游客的在线时间以离开首页为准,如点击登陆按钮
